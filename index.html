What When Where How

A few days before Christmas I got lucky with a reply from trumedianetworks.com after applying for the opportunity: SENIOR  SOFTWARE  ENGINEER
The reply from trumedianetworks included an attached .zip file with instructions in PDF and 3 type json data files.
The task: display the 3 NFL QBs on separate pages with accompanying stats.
Think time: off and on over the holidays.
Coding time: about 1 day.
Tools:

  • Pro Football reference
  • Chrome extension JSON editor online.
  • Tom Brady Stats
  • CSS griding cites
  • Various HTML5 references and usurped css from: trumedianetworks
  • My favorite CSS reference
  • IDE: WebStorm
  • Repo: My github
  • Host @: David Brown
The design: a basic HTML5 SPA.
Usage: click the link NFL QBs to redirect to the tabulated statistics page.
To see QB stats pull down and select 1 of 3 QBs corresponding to each json data file and click: HUT!  HUT!
The CLOSE button removes the tabulated data and clears the session storage.
With no QB selected from the dropdown click the HUT!  HUT! button to clear the session storage.
The remaining navigation links are hopefully self explanatory.
